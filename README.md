</div>

BS Data Science undergraduate at **FAST-NU Lahore** (CGPA 3.3/4.0), specializing in the intersection of statistical machine learning, predictive modeling, and production-grade web engineering. I build end-to-end systems — from feature-engineered ML pipelines (Scikit-learn, Pandas, NumPy) to full-stack platforms (React, Node.js, PostgreSQL) — with a focus on correctness, interpretability, and clean architecture.

On the data side, my work spans exploratory data analysis, supervised regression and classification, model evaluation (R², MAE, RMSE), and visualization (Matplotlib, Seaborn). On the engineering side, I architect RESTful APIs, relational database schemas, and component-driven frontends. I also write low-level systems code in C, C++, and x86 Assembly.

Currently leading the **PR Department at SOFTEC** — FAST Lahore's flagship national computing competition — managing corporate partnerships and communication strategy for 1000+ attendees.
