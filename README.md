### Projects

**[Brain Tumor AI Detection System](https://github.com/khushbakt0/detect-brain-tumour-ai)**
&nbsp;`Python` `TensorFlow` `EfficientNetB0` `Grad-CAM` `Streamlit`

End-to-end medical imaging classifier on brain MRI scans. Transfer learning with EfficientNetB0, custom classification head, and Grad-CAM heatmaps that show *where* in the scan the model looked — not just what it decided. Built because black-box medical AI is useless.

**[Udhaari — Peer-to-Peer Lending Platform](https://github.com/duasarfraz3114/Udhaari-fse)**
&nbsp;`React` `Node.js` `Express` `SQL`

Full-stack P2P lending platform. Complete web development stack — React frontend, Node/Express REST API, SQL database with a state-machine booking lifecycle, JWT auth, and role-based access for borrowers and lenders. Did manual research into how P2P lending works, loan risk, and how platforms like this actually operate in Pakistan's fintech space.

**[Air Quality Index Predictor](https://github.com/khushbakt0/air-quality-app)**
&nbsp;`Python` `Scikit-learn` `Pandas` `Matplotlib`

KNN-based ML model predicting air quality index from environmental sensor data. Full pipeline from raw data → EDA → preprocessing → model training → evaluation with R², MAE, and RMSE. Achieved 85% accuracy.

**[Solar Panel Web Platform](https://github.com/khushbakt0/solar-panel-web)**
&nbsp;`HTML` `CSS` `JavaScript`

Full web development project built around the solar energy sector. Researched how solar panel systems work, energy output calculations, and how solar businesses present technical information to consumers — then built the platform around those findings. Manual domain research informed every design and content decision.

**[Tannin Studio](https://tannin-studio.netlify.app)**
&nbsp;`HTML` `CSS` — *Live*

Production website for a Lahore-based leather goods brand. Product galleries, category filters, WhatsApp/call CTAs. Zero frameworks, ships fast.
